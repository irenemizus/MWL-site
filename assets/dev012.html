Instruments Devices Device 012

Device 012:

Synchronizer for BWOs of any frequency range up to 1.2 THz.

Performs phase synchronization of BWO radiation frequency against reference signal.

Synchronization bandwidth: 0.1 - 5 MHz; Automatic search and catch range: up to 1 GHz; Input signal (IF) range 300 - 400 MHz; Input signal amplitude: from -50 dBm up to +10 dBm; Allowed external reference signals: 5 MHz, 10 MHz, 30-40 MHz, 300-400 MHz. Output signal dynamic range: ±12 V; Output impedance: 50 Ohm; Multiplier-mixer diode bias control.
